25 年秋冬学期，本教学班向 [《编译原理》课程实验](https://git.zju.edu.cn/compiler) 看齐，相比之前的版本、其他教学班主要有以下改动：

- 拆分课程文档和代码仓库
- 学生统一使用 ZJU Git 上分配的仓库进行实验
- 使用容器化的实验环境
